# 论文写作技能 - 学术写作技巧

## 高质量章节的最小标准

1. 有清晰问题定义
2. 有可复现的方法描述
3. 有可核验的安全或正确性论证
4. 有可解释的实验结果

## 你的论文适配结构（当前）

1. 绪论：背景、现状、问题、贡献
2. 可验证资格检验机制（MHT）
3. 搜索结果正确性验证（嵌入式承诺）
4. 实验评估与分析
5. 总结与展望

## 学术表达规范

1. 用术语，不用口语
2. 先定义再使用符号
3. 同一术语在全文保持一致（如“资格检验”“可验证性”）
4. 每个关键结论都给出处或推导

## 常见失误清单

1. 相关工作只罗列，不比较
2. 创新点描述与正文实现脱节
3. 实验只报数字，不解释原因
4. 结论写了“计划做”，但正文没实现

## 自检清单（提交前）

1. 所有 `TODO` 是否清零
2. 引用键是否全部存在于 `ref/thesis.bib`
3. 章节间术语与符号是否一致
4. 图表是否都有正文解释
