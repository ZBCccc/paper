# CLAUDE.md - AI 总指南

## 系统概述

这是一个基于 Claude Code 的个人工作系统，旨在帮助你：

1. 建立长期记忆，让 AI 记住你的工作风格和上下文
2. 实现学术思路复用，避免重复劳动
3. 沉淀方法论，形成可复用的工作流程
4. 标准化写作，提升效率和质量

## 个人定位

请在 `个人定位.md` 文件中详细描述：
- 你是谁
- 你的专业领域
- 你希望如何使用这个系统

## 工作流程

### 论文写作流程

1. **想法收集**：在 `01-论文生产/选题管理/00-选题记录.md` 中记录所有灵感和想法
2. **待深化**：将有潜力的选题移到 `01-待深化` 文件夹，进行初步研究
3. **待发布**：完成的论文放在 `02-待投稿` 文件夹，准备发布
4. **已发布**：发布后的论文归档到 `03-已投稿` 文件夹，并记录数据

### 参考资料管理

- **核心论文库**：记录领域内的核心概念和定义，以及经典学术论文
- **相关工作库**：收集相同方向的相关工作，分析它们的优缺点
- **基础库**：整理基础知识和方法论，形成自己的知识体系

## 使用说明

### 如何与 Claude Code 交互

1. 保持上下文连贯性：每次对话前，先参考相关文件
2. 提供明确指令：说明你的需求、目标和限制条件
3. 及时更新文件：完成工作后，将成果记录到对应的位置
4. 定期回顾：每周检查系统的使用情况，进行必要调整

### 文件更新频率

- 每日：更新选题记录、灵感和素材
- 每周：整理已完成的工作，更新记忆库
- 每月：回顾系统结构，根据需要调整

## 系统维护

### 检查清单

1. 找东西时，第一反应是否与实际位置一致？
2. 是否有超过一个月未打开的文件夹？
3. 新内容放入时是否会犹豫？

如果有以上问题，说明系统需要调整优化。
